# RoamRides-Two-wheelers-website

• Designed and developed a responsive front-end e-commerce web application for two-wheelers, including bikes,
scooters, electric vehicles, and accessories, using modern web development practices.

• Implemented dynamic product listing, category-based browsing, detailed product pages, and an interactive
shopping cart to enhance user experience and usability.

• Applied client-side scripting to manage user interactions, cart state, and real-time UI updates.

• Optimized UI components for performance, accessibility, and cross-browser compatibility using vanilla JavaScript and
modular CSS styling.

Tech Stack: HTML5, CSS3, JavaScript, Responsive Web Design, DOM Manipulation
